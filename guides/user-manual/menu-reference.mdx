# Menu Functions Reference

<Note>
  All budget functions are organized in the menu bar for easy access. This guide
  explains each menu and its functions.
</Note>

## üè¶ Budget Menu

<AccordionGroup>
  <Accordion title="Import Bank Transactions" icon="file-import">
    - Access: Click "Budget > Import Bank Transactions"
    - Purpose: Import new transactions from bank statements
    - Steps:
      1. Download your bank statement
      2. Click menu option
      3. Follow the import wizard
      4. Review and confirm import
  </Accordion>

  <Accordion title="Show Reconciliation Dashboard" icon="check-double">
    - Access: Click "Budget > Show Reconciliation Dashboard"
    - Purpose: Match and verify transactions
    - Features:
      - Transaction matching
      - Category verification
      - Split transaction handling
      - Balance confirmation
  </Accordion>
</AccordionGroup>

## üìÖ Period Processing Menu

<AccordionGroup>
  <Accordion title="End of Month Process" icon="calendar-check">
    - Access: Click "Period Processing > Run End of Month Process"
    - Purpose: Close the monthly books
    - Requirements:
      - All transactions reconciled
      - Categories verified
      - Balances checked
  </Accordion>

  <Accordion title="Verify Setup" icon="gear-check">
    - Access: Click "Period Processing > Verify Setup"
    - Purpose: Check system configuration
    - Checks:
      - Category setup
      - Account mappings
      - Formula integrity
  </Accordion>
</AccordionGroup>
