    Version Recovery - Quick Start Guide

    üéØ What Is This?

    If something goes wrong with your budget spreadsheet, you can restore it to a previous version. This guide shows you how to do it safely.

    Important: Always Use the Smart Recovery Assistant

    The easiest and safest way to restore your spreadsheet is using the Smart Recovery Assistant. It guides you through every step and prevents mistakes.

    üöÄ Quick Start: Restore Your Spreadsheet

    Step 1: Open the Smart Recovery Assistant

    1. Click Settings in the menu bar
    2. Click üîÑ Version Recovery
    3. Click üß† Smart Recovery Assistant

    The assistant will open and guide you through the process.

    Step 2: Tell the Assistant When the Problem Happened

    The assistant will ask: "When did the problem occur?"

    Choose the option that best matches:

    • Just now - Problem happened very recently
    • Today - Problem happened today
    • Yesterday - Problem happened yesterday
    • This week - Problem happened this week
    • I don't know - Not sure when it happened

    Click the button that matches your situation.

    Step 2: Review the Guidance

    After selecting when the problem occurred, the assistant will show you:

    • ‚úÖ Instructions for finding the version in Version History
    • ‚úÖ How to use the Version Finder tool
    • ‚úÖ How to check property sync before restoring
    • ‚úÖ Links to helpful tools

    From Step 2, you can:

    • Click "Open Version Finder" to see available property snapshots
    • Click "Check Property Sync" to verify a version is safe to restore
    • Click "Back" to change your time selection

    Step 3: Restore the Spreadsheet Version

    1. Click "Open Version History" (or go to File > Version History > See version history)
    2. Find the version you want to restore
    3. Click "Restore this version"
    4. Confirm the restoration

    After restoring, the assistant will automatically move to Step 3.

    Step 3 (After Restore): Restore Properties (If Needed)

    After you've restored the spreadsheet version, the assistant will show Step 3 with:

    • ‚úÖ Instructions for restoring properties
    • ‚úÖ Button to restore properties from archive
    • ‚úÖ Option to check property sync again

    If the assistant tells you to restore properties:

    1. Click "Restore Properties"
    2. Enter the version date you restored (or click "Skip" to find best match)
    3. Confirm the restoration

    That's it! Your spreadsheet and properties are now synchronized.

    üìã Common Scenarios

    Scenario 1: I Made a Mistake Today

    What to do:

    1. Open Smart Recovery Assistant
    2. Select "Just now" or "Today"
    3. Follow the assistant's instructions
    4. Restore to a version from earlier today

    Result: Your spreadsheet goes back to how it was before the mistake.

    Scenario 2: I Need to Go Back to Before End of Month

    What to do:

    1. Open Smart Recovery Assistant
    2. Select "Yesterday" or "This week" (depending on when EOM ran)
    3. Review the guidance in Step 2
    4. Restore the version from before EOM (File > Version History)
    5. Restore properties when Step 3 appears - the system will automatically find the correct PRE-EOM properties

    Result: Your spreadsheet and properties go back to the previous month (correct state).

    Scenario 3: I'm Not Sure When the Problem Happened

    What to do:

    1. Open Smart Recovery Assistant
    2. Select "I don't know"
    3. Use the Version Finder tool to see available versions
    4. Check the property sync for versions you're considering
    5. Choose a version that's safe to restore

    Result: You can browse and find the right version to restore.

    üõ†Ô∏è Tools Available

    üß† Smart Recovery Assistant (Recommended)

    What it does:

    • Guides you through the entire recovery process
    • Shows you exactly what to do at each step
    • Prevents mistakes
    • Makes recovery easy and safe

    When to use: Always! This is the easiest way to restore your spreadsheet.

    How to access: Settings > Version Recovery > Smart Recovery Assistant

    üîç Version Finder

    What it does:

    • Shows you all available property snapshots
    • Helps you find versions that are safe to restore
    • Shows when properties were exported

    When to use: When you want to see what versions are available before restoring.

    How to access: Settings > Version Recovery > Version Finder

    ‚úì Check Property Sync

    What it does:

    • Checks if a version is safe to restore
    • Shows if properties match the version
    • Warns you if there's a problem

    When to use: Before restoring a version, to make sure it's safe.

    How to access: Settings > Version Recovery > Check Property Sync

    üì• Restore Properties

    What it does:

    • Restores properties to match your restored spreadsheet
    • Makes sure everything is synchronized
    • Prevents system errors

    When to use: After restoring a spreadsheet version, if the assistant tells you to.

    How to access: Settings > Version Recovery > Restore Properties

    ‚ö†Ô∏è Important Warnings

    Don't Restore Without Checking First

    If you restore a version that doesn't have matching properties, your system will break.

    What happens: - Spreadsheet shows one month - Properties think it's a different month - System becomes unusable

    Solution: Always use the Smart Recovery Assistant or check property sync first.

    Always Use the Smart Recovery Assistant

    It prevents mistakes and guides you safely through the process.

    What it does: - Checks if restore is safe - Shows you what to do - Prevents system breakage

    Result: Safe, easy recovery every time.

    üÜò What If Something Goes Wrong?

    Problem: "No matching snapshot found"

    What it means: The version you want to restore doesn't have matching properties.

    What to do:

    1. Don't restore that version - it will break your system
    2. Choose a different version that has a matching snapshot
    3. Use the Version Finder to see which versions are safe
    4. Contact support if you need help

    Problem: "Properties are out of sync"

    What it means: The properties don't match the restored spreadsheet.

    What to do:

    1. Use Restore Properties to fix it
    2. The system will find the correct properties
    3. Confirm the restoration
    4. Your system will be synchronized

    Problem: I Can't Find the Version I Want

    What to do:

    1. Use the Version Finder to see available versions
    2. Check the property export timeline
    3. Look for versions with matching snapshots
    4. If you can't find it, the version might not be safe to restore

    üìñ Step-by-Step Example

    Example: Restoring After a Mistake

    Situation: You accidentally deleted some data this morning.

    Step 1: Open Smart Recovery Assistant

    • Settings > Version Recovery > Smart Recovery Assistant

    Step 1 (continued): Select "Today"

    • Click the "Today" button

    Step 2: Review the guidance

    • The assistant shows instructions for:
    • Finding versions from today
    • Checking if they're safe to restore
    • Using the Version Finder tool
    • You can click "Check Property Sync" to verify a version is safe

    Step 3: Restore the version

    • Click "Open Version History" (or go to File > Version History > See version history)
    • Find a version from before the mistake
    • Click "Restore this version"
    • Confirm the restoration

    Step 3 (After Restore): Restore properties (if needed)

    • The assistant automatically moves to Step 3 after you restore
    • If it tells you to restore properties, click "Restore Properties"
    • Enter the version date (or click "Skip" to find best match)
    • Confirm the restoration

    Result: Your spreadsheet is back to how it was before the mistake!

    • Check property sync before restoring - prevents problems

    üéì Understanding the System

    What Are Properties?

    Properties are settings that control how your budget works:

    • Current month
    • Month statuses
    • System configuration

    Why they matter: If properties don't match your spreadsheet, the system breaks.

    How we fix it: We store snapshots of properties and restore them when needed.

    What Is a Snapshot?

    A snapshot is a copy of your properties at a specific time.

    When snapshots are created:

    • Before End of Month (EOM) processing
    • After End of Month (EOM) processing

    Why they're important: They let us restore properties to match your restored spreadsheet.

    What Is Version History?

    Version History is Google Sheets' built-in feature that saves copies of your spreadsheet over time.

    What it saves:

    • ‚úÖ Spreadsheet data
    • ‚úÖ Formulas
    • ‚úÖ Structure

    What it doesn't save:

    • ‚ùå Properties (we handle this separately)

    Why we need both: To restore everything correctly, we need both the spreadsheet version AND the matching properties.

    üìû Need Help?

    If You're Stuck

    1. Use the Smart Recovery Assistant - it guides you through everything
    2. Check the Version Finder - see what versions are available
    3. Check Property Sync - verify if a version is safe to restore
    4. Contact support - if you need additional help

    Common Questions

    Q: Can I restore to any version?
    A: Only versions that have matching property snapshots. The system will tell you if a version is safe to restore.

    Q: What if I restore the wrong version?
    A: You can restore again to a different version. Just use the Smart Recovery Assistant again.

    Q: Will I lose my current data?
    A: Yes, restoring replaces your current spreadsheet with the restored version. Make sure you want to restore before doing it.

    Q: How far back can I restore?
    A: You can restore to any version in your version history, but only if it has a matching property snapshot. The system will tell you which versions are safe.

    ‚úÖ Quick Checklist

    Before restoring, make sure you:

    • Opened the Smart Recovery Assistant (Settings > Version Recovery > Smart Recovery Assistant)
    • Selected when the problem occurred (Step 1)
    • Reviewed the guidance shown (Step 2)
    • Checked property sync (if recommended, from Step 2)
    • Opened Version History and found the version to restore
    • Confirmed it's safe to restore (using Check Property Sync if needed)
    • Restored the spreadsheet version (File > Version History > Restore this version)
    • Restored properties (if Step 3 tells you to)
    • Verified everything works correctly

    4. Restore the version (File > Version History > Restore this version)
    5. Restore properties (Step 3, if needed) - assistant guides you
